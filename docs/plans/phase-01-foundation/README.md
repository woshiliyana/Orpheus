This directory stores work-package plans for `Phase 1: Experience Foundation + Platform Shell`.

Use filename format `NN-short-name.md`.

Typical work-package types here include:

1. `brand-ux-shell`
2. `hero-claim-funnel`
3. `tool-page-template`
4. `seo-landing-template`
5. `platform-shell-foundation`

Keep UX shell work separate from platform shell work when possible.
