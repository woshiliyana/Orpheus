# Prueba breve de narracion

Esta es una prueba corta para confirmar que la narracion en espanol acepta texto con puntuacion y conserva el contenido original. El objetivo es revisar la ruta del proveedor, no evaluar una pieza larga.

Primero, el sistema debe leer el texto sin reescribirlo. Segundo, debe agregar pausas suaves entre parrafos. Tercero, debe guardar evidencia suficiente para revisar el resultado despues.
