# Plan de lanzamiento

El equipo tiene una meta esta semana: convertir una ruta de narracion que ya funciona en algo que un creador pueda confiar. La carga del guion debe sentirse tranquila, predecible y honesta sobre lo que esta ocurriendo.

## Que cambia para el creador

Pega el guion completo una sola vez. Elige una voz. Presiona generar, y recibe el audio final con tiempos listos para subtitulos, sin dividir el texto manualmente.

- Primero, el sistema comprueba si el guion es legible.
- Luego, conserva intactas las palabras originales.
- Despues, agrega solo pausas estructurales entre secciones.
- Finalmente, guarda evidencia para que un revisor pueda ver lo que paso.

## Lo que todavia no vamos a prometer

No vamos a declarar compatibilidad multilingue completa hoy. No vamos a exponer controles del proveedor en el navegador. No vamos a vender funciones publicas amplias de exportacion antes de que la primera puerta sea estable.

Esa es la disciplina: menos promesas, mejores recibos, y una narracion que suene deliberada en vez de apresurada.
